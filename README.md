# WeatherParser

Приложение командной строки для получения данных о погоде в городах России с сервиса Метеосервис путем парсинга html страниц.
При запуске приложения задается список городов требуемых для парсинга, данные для которых обновляются ежеминутно и сохраняются в папку /Data в виде текстовых файлов с названием городов.
Приложение сохраняет лог работы в файл Log.txt

Пример: WeatherParser.exe Москва Питер Киев
Где Москва Питер Киев  - города, кэш погоды которых требуется получить.
